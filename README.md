# Building & Running
1. Compile `Client.c` & `Server.c` with the `Makefile`.
```
$ make
```
2. Run server with `make server`.
```
$ make server
```
3. Run client with `make client`.
```
$ make client
```
4. Clean up with `make clean`.
```
$ make clean
```
